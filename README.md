# 🌯 Lemor Shawarma

A mockup website for **Lemor Shawarma** — an authentic Middle Eastern shawarma restaurant.

## Pages & Sections

| Section | Description |
|---|---|
| **Hero** | Eye-catching landing banner with call-to-action buttons |
| **Features Strip** | Quick highlights: freshly grilled, fast delivery, halal certified |
| **Menu** | Tabbed menu with Wraps, Plates, and Sides & Drinks |
| **About** | Restaurant story and key statistics |
| **Testimonials** | Customer reviews |
| **Contact / Order** | Location info and online order form |
| **Footer** | Navigation links and social media |

## Tech Stack

- Plain **HTML5**, **CSS3**, and vanilla **JavaScript** — no build tools required.
- Open `index.html` directly in any modern browser to view the mockup.

## Running Locally

```bash
# Simply open the file in a browser
open index.html
# — or —
npx serve .
```
